How it works

Purpose-built detection so nothing slips by.

Terminal Meme Error Sounds leans on VS Code’s terminal APIs and layered fallbacks to make sure every failure is heard.

01

Listen to your shells

Hooks into shell execution end events to watch exit codes in every terminal tab.

02

Scan for bad output

Pattern matching for common error strings catches failures even when exit codes lie.

03

Cover tasks & closes

Build/test task process failures and non-zero exits on close both fire the same audio cue.

04

Play a random meme

Picks a track, respects your cooldown, and logs to “Terminal Meme Sounds” for easy debugging.
